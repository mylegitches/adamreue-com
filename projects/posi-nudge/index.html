PosiNudge

Mental Wellness Through Positive Nudges

Personal Project 2024

Overview

PosiNudge is a React Native mobile application that promotes mental wellness by detecting negative language patterns and providing gentle, timely reminders to encourage more positive communication. Using natural language processing and sentiment analysis, the app monitors text input and sends contextual notifications when negative language exceeds user-defined thresholds.

The app focuses on behavioral psychology principles, using positive reinforcement and gentle nudges rather than punishment to help users develop healthier communication habits and improve their mental well-being.

Key Features

🧠

NLP Analysis

Real-time natural language processing to detect negative sentiment patterns in text input

📱

Smart Notifications

Gentle, contextual reminders sent when negative language thresholds are exceeded

📊

Progress Tracking

Detailed analytics showing communication patterns and improvement over time

🎯

Customizable Thresholds

User-defined sensitivity levels and notification preferences
