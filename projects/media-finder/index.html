Media Finder

Intelligent Content Discovery Platform

Personal Project 2023

Overview

Media Finder is a Python application that helps users discover new movies, TV shows, music, books, and other media content. By entering information about content they already enjoy (actors, directors, bands, authors), the tool queries multiple online databases to find similar works and recommendations, with seamless integration into popular media management platforms.

Key Features

🔍

Multi-Source Search

Queries multiple online databases including TMDB, MusicBrainz, and Goodreads for comprehensive results

🎯

Smart Recommendations

Uses collaborative filtering and content-based algorithms to suggest relevant content

📊

*ARR Integration

Direct integration with Radarr, Sonarr, Lidarr, and Readarr for automated media management

🌐

Cross-Media Discovery

Finds connections between different media types (books to movies, music to films, etc.)

Technology Stack
